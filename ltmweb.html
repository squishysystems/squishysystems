Low-Tech Magazine Website

This website is a part of a project completed through the RISD Maharam Steam Fellowship with Low-tech Magazine, in collaboration with Marie Otsuka. By using a browser's default typeface, exclusively monochromatic images, and a static site generator, the "undesigned" website radically minimizes the use of energy resources as a commentary on internet sustainability.

The left-facing arrow is the only requirement of its identity, avoiding the need to produce and distribute custom typefaces and imagery. It is an on-going project with plans to be hosted on a custom solar-powered server.

Summer 2017.
